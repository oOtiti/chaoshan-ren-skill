# 粤东潮汕文化参考

这份资料用于帮助 Codex 写出更稳、更有地方感的潮汕文化内容。它不是完整民俗志，也不能替代当地长辈、村社、宗亲组织、庙宇理事会或专业研究者的具体判断。

## 范围

- “潮汕”优先指粤东潮汕文化圈，重点覆盖汕头、潮州、揭阳及其相关县区、乡镇和海内外潮人社群。
- “潮州”既可能指今天的潮州市，也可能指历史上的潮州府、潮州话、潮剧、潮州音乐、潮州菜等传统称谓。不要把“潮州”机械等同于“整个潮汕”。
- “潮汕人”不是单一性格标签。写作时要保留城乡、家庭、姓氏、行业、移民经历、海外社群和代际变化。
- 如果用户说“珠东潮汕文化”，一般先按“粤东潮汕文化圈”理解；如果语境确实指珠三角东部或其他范围，再跟随用户上下文调整。

## 核心文化线索

- 工夫茶：既是饮茶方式，也是待客、聊天、敬长辈和维系关系的日常礼仪。写作时强调“细致、讲究、有人情”，不要只写茶具动作。
- 潮汕话 / 潮州话 / 潮语：属于闽南语系潮汕片，是潮剧、潮州音乐、歌册、日常称谓和海外潮人认同的重要载体。不要随意编造读音和罗马字。
- 潮剧：以潮汕方言演唱，是潮汕地方文化和海外潮人乡音记忆的重要载体。适合用于写乡音、舞台、庙会、侨社和文化传承。
- 英歌：潮汕民间广场表演艺术，常见于潮阳、潮南、普宁等地，融合戏剧、舞蹈、武术、锣鼓和《水浒》人物想象。写作时不要只说“战舞”，还要看到村社组织、训练传承和节庆场景。
- 潮汕菜 / 潮州菜：重鲜、重汤、重刀工火候，讲究蘸料、卤水、海鲜、粥饭和粿品。可以写牛肉火锅，但不要把潮汕饮食缩成牛肉火锅。
- 粿：米制食物大类，连接节令、祭祀、家庭劳动和街市小吃。红桃粿、鼠壳粿、无米粿、菜头粿、甜粿等都适合写家庭记忆。
- 潮汕古建筑：可写下山虎、四点金、嵌瓷、木雕、石雕、灰塑、彩绘、祠堂和侨宅。重点是礼制空间、审美和工艺，不要只写“豪华”。
- 侨批和侨乡：潮汕是著名侨乡，侨批把汇款、家书、亲情和跨海生计连在一起。写海外潮人时，要写“出海谋生、家书往返、社团互助、乡音乡情”，不要写成单纯商业成功学。

## 风俗与节庆

- 出花园：潮汕语境中的成人礼，常与十五岁、沐浴、换新衣、红色元素、祭拜“公婆母/花公花母”、吃象征性食物、感谢父母等叙事相连。各地流程差异大，不能写成统一标准。
- 行彩桥：春节或元宵前后的民俗活动之一，在揭阳古城等地常作为年俗和城市文化记忆出现。写作时可强调新年祈愿、人群流动、古城街巷和节庆氛围。
- 七样羹 / 七样菜：多与农历正月初七相关，用七种蔬菜承载勤劳、聪明、长久、和合等谐音或象征。具体菜名会随地方和家庭变化。
- 营老爷 / 闹热 / 游神：潮汕部分地区常见的社区性节俗，和地方神明、村社秩序、锣鼓、标旗、英歌、潮剧、祈福等相连。要用尊重、中性的语言，不要猎奇化，也不要写成全潮汕统一时间表。
- 清明、冬至、中秋、春节：常和团圆、祭祖、做粿、拜访亲友、庙会演出等相连。需要具体流程时，要让用户提供地方和家庭习惯。

## 礼仪与家庭表达

- 待客：工夫茶是最稳的入口。可以写“先烧水泡茶、递给长辈和客人、边饮边谈”，但不要写成固定仪轨。
- 家庭：潮汕家庭叙事常涉及长辈、祖厝、祠堂、族亲、节令团聚和外出谋生。写时要温和，不要把宗族说成压迫或浪漫化的唯一解释。
- 婚嫁：可以概括茶礼、敬亲、礼数、喜糖喜饼、宴席和亲友往来，但不要编完整流程。
- 祭祖：可以写慎终追远、感念祖先、家族记忆和节令秩序；不要给出具体供品、方位、咒语或禁忌清单，除非用户提供当地做法且只是整理用户材料。

## 表达风格

- 好的写法：从生活画面进入，例如“茶盘一摆，长辈先坐下，话也慢慢开了”“正月锣鼓一响，村里的年味就从巷口涌出来”“一个红桃粿，包着的不只是米香，也是一家人过节的手势”。
- 少用大词：不要一上来就“千年古韵、源远流长、东方神秘力量”。可以写得有诗意，但要落在具体场景。
- 公开文案要稳：先说明范围，再说明习俗，再说明差异，最后说明今天的意义。
- 家族文案要暖：少解释，多敬意；少炫耀，多感恩；少“潮汕人都如何”，多“我们家、我们这一带、许多潮汕家庭会如何”。

## 不要这样写

- 不要把潮州当成全部潮汕，也不要为了纠偏而贬低潮州。
- 不要说“潮汕人都经商”“潮汕人都很宗族”“潮汕女人都如何”“潮汕家庭都必须拜老爷”。
- 不要把民俗写成迷信奇观，也不要把信俗流程写成网上通用教程。
- 不要编造潮汕话读音、祖训、禁忌、古法来源或“老一辈都这么说”。
- 不要把牛肉火锅、英歌、工夫茶当成潮汕文化的全部。
